# Fake News Review Site

This static site mirrors the interaction style of `dental.scoring.github.io`, but is adapted for fake news review.

## Rebuild

Run:

```powershell
python .\build_fake_news_review_site.py
```

The script reads `sample_bundle_50x3`, generates `fakenews.scoring.github.io\data.js`, and copies the review images into `fakenews.scoring.github.io\assets\images`.
